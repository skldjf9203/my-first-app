# my-first-app

A simple Node.js Express application.

## Requirements

- Node.js (v14 or higher)
- npm

## Run Instructions

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

3. Open your browser and visit [http://localhost:3000](http://localhost:3000)

   You should see: `Hello from Coolify 🚀`
